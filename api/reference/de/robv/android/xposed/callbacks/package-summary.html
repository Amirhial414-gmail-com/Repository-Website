package

de.robv.android.xposed.callbacks

Contains the base classes for callbacks.

For historical reasons, XC_MethodHook and XC_MethodReplacement are directly in the de.robv.android.xposed package.

Interfaces

IXUnhook<T> Interface for objects that can be used to remove callbacks.
