IXUnhook

de.robv.android.xposed.callbacks.IXUnhook<T>
Known Indirect Subclasses
XC_LayoutInflated.Unhook, XC_MethodHook.Unhook
XC_LayoutInflated.Unhook An object with which the callback can be removed. 
XC_MethodHook.Unhook An object with which the method/constructor can be unhooked. 

Class Overview

Interface for objects that can be used to remove callbacks.

Just like hooking methods etc., unhooking applies only to the current process. In other process (or when the app is removed from memory and then restarted), the hook will still be active. The Zygote process (see IXposedHookZygoteInit) is an exception, the hook won't be inherited by any future processes forked from it in the future.
